Introduction

Python is a high-level, interpreted, and general-purpose programming language known for its readability and simplicity. It is designed to be easy to understand and fun to use, making it an excellent choice for beginners as well as experienced programmers.

Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming. Its extensive standard library and community-contributed modules make it highly versatile for various applications, from web development to data analysis and artificial intelligence.

What You Should Already Know

This guide assumes you have the following basic background:

  • A general understanding of programming concepts.
  • Familiarity with basic mathematical operations.
  • An understanding of how to use a text editor or an integrated development environment (IDE).
Python and Other Languages

While Python shares similarities with other programming languages such as Java and C++, it is distinct in several ways:

  • Dynamic Typing: Python uses dynamic typing, which means you don't need to declare variable types explicitly. The interpreter infers the type at runtime.
  • Indentation: Python uses indentation to define code blocks, eliminating the need for braces or keywords.
  • Rich Standard Library: Python has a vast standard library that supports many common programming tasks, from file I/O to web development.
Hello World

To get started with Python, open your Python interpreter or an IDE and write your first "Hello, World!" code:

print("Hello, World!")

When you run this code, it will display the message "Hello, World!" on the screen.

Variables

In Python, you use variables as symbolic names for values in your application. Variable names, known as identifiers, follow certain rules:

  • A Python identifier must start with a letter (a-z, A-Z) or an underscore (_).
  • Subsequent characters can also be digits (0-9).
  • Python is case-sensitive, meaning that `myVariable` and `myvariable` are different identifiers.
Declaring Variables

You can declare a variable by simply assigning a value to it:

x = 42
y = "Hello"

There is no need for a declaration keyword; the variable type is inferred from the value assigned.

Variables Scope

Variables can have different scopes:

  • Global Variables: Declared outside of any function, accessible throughout the script.
  • Local Variables: Declared within a function, accessible only within that function.
Example: 
global_var = "I am global"

def my_function():
    local_var = "I am local"
    print(global_var)
    print(local_var)

my_function()
# print(local_var)  # This would raise an error
Constants

Python does not have a built-in constant type, but by convention, you can use uppercase letters to denote constants:

PI = 3.14

While the value of `PI` can technically be changed, it is considered good practice not to do so.

Data Types

Python has several built-in data types:

  • Numeric: `int`, `float`, `complex`
  • Boolean: `True` and `False`
  • Sequence: `list`, `tuple`, `range`
  • Text: `str`
  • Mapping: `dict`
  • Set: `set`, `frozenset`
  • Binary: `bytes`, `bytearray`, `memoryview`
Control Structures

If...Else Statement

Use the `if` statement to execute a block of code if a condition is true:

if condition:
    # code to execute if condition is true
else:
    # code to execute if condition is false

You can also add `elif` for multiple conditions:

if condition1:
    # code for condition1
elif condition2:
    # code for condition2
else:
    # code if neither condition is true

While Loop

A `while` loop executes a block of code while a specified condition evaluates to true:

n = 0
while n < 3:
    n += 1
    print(n)
Function Declarations

A function in Python is defined using the `def` keyword:

def square(number):
    return number * number

You call a function by using its name followed by parentheses:

result = square(4)  
    # result will be 16

Parameters are passed by reference for mutable types (like lists), and by value for immutable types (like integers).
